## 🎮 **COMO USAR**

### 📥 **Instalação**
1. **Baixe** o arquivo `KeyDrop_Bot_v2.0.3_Fixed.zip` desta release
2. **Extraia** o conteúdo para uma pasta de sua preferência
3. **Execute** `KeyDrop_Bot_Moderno.exe` ou `KeyDrop_Bot_Classico.exe`
